# Archive

This directory contains old/superseded files from the Lowe HTA model development process.

## Directory Structure

### `/scripts/`
Contains old calibration, testing, debugging, and analysis scripts:
- `calibrate_*.py` - Parameter calibration scripts
- `test_*.py` - Model testing and validation scripts
- `debug_*.py` - Debugging scripts
- `diagnose_*.py` - Diagnostic scripts
- `validate_*.py` - Validation scripts
- `example_*.py` - Example analysis scripts
- `generate_*.py` - Figure generation scripts
- `plot_*.py` - Plotting scripts

### `/figures/`
Contains old figures and visualizations generated during model development:
- Age-varying rates figures
- eGFR trajectories
- Cost-effectiveness planes
- Treatment scenarios

### `/old_outputs/`
Contains superseded output files from previous model runs:
- `.csv` files with old analysis results
- `model_summary.txt` - Old model summary

### `/old_docs/`
Contains outdated documentation:
- `DELIVERABLES_SUMMARY.md`
- `FIGURES_README.md`
- `MULTI_ORGAN_DISEASE_JUSTIFICATION.md`

## Note

These files are kept for reference but are no longer actively used.
Current outputs are generated in the `/outputs/` directory.
Current model is `markov_cua_model.py` in the parent directory.
